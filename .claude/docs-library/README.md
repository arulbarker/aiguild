# docs-library

Dokumentasi teknis project AI Guild — digunakan Claude Code sebagai referensi saat coding.

## Aturan folder ini
- Satu topik = satu file, nama kebab-case
- Setiap file baru wajib didaftarkan di CLAUDE.md
- Selalu update jika ada perubahan kode yang relevan
- File tidak relevan → arsip ke `.claude/archive/`, jangan hapus langsung

## Cara sorting file .md baru
Tanya: "Siapa yang butuh file ini?"
- Claude butuh untuk project ini → simpan di sini
- Claude butuh di semua project → pindah ke `~/.claude/templates/`
- User/customer butuh → simpan di `docs/` root project
